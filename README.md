# word_search_game

## Backend setup
```
cd ./backend
```
- Install dependencies
```
npm install
```
- Run (by default on port 3001)
```
npm start
```

## Frontend setup
```
cd ..
```
- Install dependencies
```
npm install
```
- Compiles and hot-reloads for development
```
npm start
```
- Compiles and minifies for production
```
npm run build
```
